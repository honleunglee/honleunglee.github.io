From the Fermat-Torricelli problem

July XX, 2026

The Fermat-Torricelli problem [1] is a classical geometry problem raised by Fermat (1605-1665). Given three points on a plane forming a triangle, the problem is to find a point on the plane so that the sum of the three distances from each given point to this point is the smallest. The solution is called the Fermat-Torricelli point.

Expressing in modern language, given three non-collinear points $a_1,a_2,a_3\in\mathbb{R}^2$, the Fermat-Torricelli problem is to find $x_0\in \mathbb{R}^2$ so that $x_0 = {\rm argmin}_{x\in\mathbb{R}^2} \varphi(x)$ where the cost function $\varphi$ is given by $$ \varphi(x) = \|x-a_1\|_2 + \|x-a_2\|_2 + \|x-a_3\|_2, \text{ for all } x\in \mathbb{R}^2. $$ As $\varphi$ is a convex coercive function on $\mathbb{R}^2$, we know the problem has a unique solution which is the Fermat-Torricelli point.

The Fermat-Torricelli point $x_0$ can be described as follows. If any one of the interior angles of the triangle, say the angle between $a_2-a_1$ and $a_3-a_1$, is at least $120$ degrees, then $x_0=a_1$. Otherwise, $x_0$ is a point so that the angle between $a_i - x_0$ and $a_j - x_0$ is exactly $120$ degrees for any $1\leq i < j \leq 3$.
